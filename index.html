Outils piano

1. Devine le dessin de l'accord

Le nom de l'accord s'affiche, clique sur les bonnes touches.

2. Devine le nom de l'accord

L'accord est dessiné sur le clavier, trouve son nom et son état.

3. Accords éligibles

Sélectionne 1 à 3 notes, découvre les accords possibles.

4. Gammes

Choisis une note, visualise sa gamme sur le clavier.

5. Cercle des quintes

Trouve les accords compatibles pour enchaîner facilement.
